# Deep Learning-Based IEEE 802.15.4z HRP UWB Radio Tomographic Imaging (RTI)

## Project Overview

Implement a complete UWB RTI system: mathematical forward model, training data generation, deep learning model (Dual-Branch Physics-Informed U-Net), training, evaluation, and visualization. All code in Python.

The proposed model combines a fixed Tikhonov reconstruction branch with a learnable fully connected branch, followed by a U-Net refinement network. Ablation studies (Tikhonov-Only, FC-Only) are conducted to verify the contribution of each component.
